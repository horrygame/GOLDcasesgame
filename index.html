📚 Пособие по программированию

Основные команды

атаковать_ближайшего() - атаковать ближайшего врага
двигаться_к(x, y) - двигаться к указанным координатам
двигаться_вперед(n) - двигаться вперед на n пикселей
двигаться_назад(n) - двигаться назад на n пикселей
двигаться_влево(n) - двигаться влево на n пикселей
двигаться_вправо(n) - двигаться вправо на n пикселей

Проверка условий

есть_враг_в_радиусе() - возвращает ИСТИНА, если враг в радиусе атаки
расстояние_до_врага() - возвращает расстояние до ближайшего врага
здоровье() - возвращает текущее здоровье юнита
координата_x() - возвращает X-координату юнита
координата_y() - возвращает Y-координату юнита

Конструкции языка

# Ветвление (условие)
если условие:
    действие
иначе:
    другое_действие
# Цикл с счетчиком
повторить 5 раз:
    действие
# Цикл с условием
пока условие истинно:
    действие
